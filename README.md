# nokhba-automation-scripts-
Automation pipeline for Nokhba e-commerce store, integrating AliExpress affiliate data processing via Make.com and n8n.
Project Overview
​This project leverages advanced no-code infrastructure to bridge e-commerce platforms with affiliate marketing systems. The pipeline automates the processing of consumer trend data and affiliate product links.
​Tech Stack
​Automation: Make.com for orchestration and data flow.
​Backend logic: n8n for handling complex task automation.
​Integration: AliExpress Affiliate API integration for automated product sourcing and link generation.
​Goals
​The primary objective of these scripts is to maintain a high-efficiency publishing pipeline, ensuring that the Nokhba storefront remains populated with high-value consumer items while minimizing manual intervention.
