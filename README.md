# 🚀 Nokhba Automation Pipeline

An autonomous marketing infrastructure designed to streamline affiliate operations, from product sourcing to data-driven content generation.

## 🏗️ Architecture Overview
The system is built on a modular automation pipeline using **Make.com** and **n8n** to minimize manual intervention.

### Core Modules
1. **Product Sourcing**: Automates retrieval of high-conversion products from AliExpress Affiliate Center.
2. **AI Analysis**: Uses Claude API to evaluate market demand, profit margins, and competition.
3. **Content Generation**: Automates multi-lingual ad copy (FR/EN/DE) for global markets.
4. **Data Sync**: Centralizes campaign performance metrics in Airtable for weekly analysis.

## 🛠️ Tech Stack
* **Orchestration**: [Make.com](https://make.com)
* **Backend Logic**: [n8n](https://n8n.io)
* **Intelligence**: [Claude API](https://anthropic.com)
* **Database**: [Airtable](https://airtable.com)
